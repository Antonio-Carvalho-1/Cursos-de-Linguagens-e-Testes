Parágrafos e quebra de linhas

Você pode escrever um parágrafo de qualquer jeito no código com varias linhas dentro da tag. Basta colocar tudo no meio das tag p e /p. Mesmo saltando várias linhas no código a página não será alterada.

Se precisar quebrar o código em algum lugar específico basta utilizar a tag "br".
Mesmo estando escrito na mesma linha do código foi, foi feito uma quebra de linha com a tag "br".

tag <br> significa quebra de linha!
